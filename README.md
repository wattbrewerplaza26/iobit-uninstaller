# IObit Uninstaller Cleanup Planner

A cleanup planning page for uninstalling apps without forgetting the boring leftovers.

It is useful for listing what you removed, what still needs checking, and which startup entries or folders deserve a second look.

The main page is here:

<p align="center">

## What you can use it for

- Uninstall notes
- Leftover review
- Startup cleanup
- Maintenance checklist

## Why this repo is small

Not every project needs a heavy codebase. This repo is here to give the topic a
clean public home on GitHub, keep the Mirrorify domain easy to find, and leave a
simple place for repository metadata.


Unofficial planner only. No proprietary software or license bypass content.

That's the whole point: clear page, clean files, no clutter.
